# Buscador de Series (TVMaze)

Lo empleado en esta práctica ha sido
- React (Hooks)
- HTML5
- La API de TVMaze provista en el enunciado de la práctica [https://www.tvmaze.com/api]

## Funcionalidades

- Buscar series por nombre o palabra clave
- Ver la info detallada de la serie
- Añadir o quitar series de la lista de favoritos
- Visualizar dicha lista de favoritos
